Main Configuration
==================

The plugin needs to know the merchant e-mail address.

::

   plugin.tx_cartpostfinance {
       spaceId = 405
       userId = 512
       secret = FKrO76r5VwJtBrqZawBspljbBNOxp5veKQQkOnZxucQ=
   }

|

.. container:: table-row

   Property
         plugin.tx_cartpostfinance.spaceId
   Data type
         string
   Description
         The Space Id for your account. You can find it in your account settings.

.. container:: table-row

   Property
         plugin.tx_cartpostfinance.userId
   Data type
         string
   Description
         The User Id for your account. You can find it in your account settings.

.. container:: table-row

   Property
         plugin.tx_cartpostfinance.secret
   Data type
         string
   Description
         The Secret for your account. You can find it in your account settings.
